BeatSaber to PowerBeatsVR Converter


This is a hacked together reverse engineered converter that takes zip files from BeatSaber and convertes them to PowerBeatsVR levels. You can fork improve the code and UI on github


For downloading songs I recommend going to beatsaver.com and downloading zip files.
For example, try downloading Under The Sea (Little Mermaid) - Samuel E. Wright


More example levels can be found here.

Place the layout files in Steam/steamapps/common/PowerBeatsVR/PowerBeatsVR_Data/Layouts.
